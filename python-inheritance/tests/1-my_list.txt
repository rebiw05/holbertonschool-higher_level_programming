Test the MyList class and its print_sorted method.

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

Test with an empty list.

>>> empty_list = MyList()
>>> empty_list.print_sorted()
[]

Test with a single element.

>>> single_list = MyList()
>>> single_list.append(10)
>>> single_list.print_sorted()
[10]

Test with duplicate elements.

>>> dup_list = MyList()
>>> dup_list.append(5)
>>> dup_list.append(2)
>>> dup_list.append(5)
>>> dup_list.append(1)
>>> dup_list.print_sorted()
[1, 2, 5, 5]
